# SKILL: hello-world

A simple greeting skill that demonstrates the basic structure of a Claude Code skill. This is perfect for learning how skills work.

## Triggers
- "say hello"
- "greet me"
- "hello world"

## Behaviour

### Step 1: Activation
When the user uses one of the trigger phrases, this skill activates.

### Step 2: Response
The skill responds with a friendly greeting and explains its purpose.

### Step 3: Demonstration
Shows the user how skills can:
- Respond to natural language
- Provide helpful information
- Demonstrate plugin capabilities
