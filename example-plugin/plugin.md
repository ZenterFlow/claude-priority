# Example Plugin

## Purpose
Demonstrate all features and best practices for Claude Code plugin development through working examples and comprehensive documentation.

## Activation Triggers
- "help with example plugin"
- "show me plugin examples"
- "how do I create a plugin?"
- "plugin development guide"

## Topics Covered
- Plugin structure and organization
- Skill development patterns
- Agent implementation
- Frontmatter and metadata
- Validation and testing
- Best practices and conventions
- Tool usage (Read, Write, Bash, etc.)
- Naming conventions
- Documentation standards

## Workflow
1. User requests plugin development help
2. Agent assesses user's experience level and goals
3. Provides relevant examples and explanations
4. Demonstrates skills in action
5. Walks through validation and testing
6. Offers hands-on practice opportunities
7. Answers follow-up questions

## Resources
- hello-world skill: Simplest possible example
- status-checker skill: Interactive skill with system checks
- example-plugin-tutor agent: Interactive teaching guide
- Validation scripts in plugin-formatter
- Template files in plugin-formatter/skills/plugin-formatter/templates
- Before/after examples in plugin-formatter/examples
