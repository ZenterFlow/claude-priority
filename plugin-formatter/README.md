# Plugin Formatter

## What This Plugin Teaches
This plugin helps developers format and validate Claude Code plugins to ensure they meet marketplace guidelines and best practices.

## Skills Included
- **plugin-formatter**: Validates plugin structure, naming conventions, metadata files, and documentation

## Agent
- **plugin-formatter-tutor**: Interactive guide for plugin formatting and validation

## How to Use
1. Navigate to your plugin directory
2. Type: "format this plugin" or "validate plugin format"
3. Follow the suggested corrections
4. Review the validation report
