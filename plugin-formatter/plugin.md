# Plugin Formatter

## Purpose
Automatically format and validate Claude Code plugins to ensure compliance with marketplace guidelines and best practices.

## Activation Triggers
- "format this plugin"
- "fix plugin structure"
- "validate plugin format"
- "check plugin compliance"

## Topics Covered
- Plugin directory structure
- JSON schema validation (plugin.json, marketplace.json)
- Naming conventions (lowercase-with-hyphens)
- Metadata completeness
- Skill frontmatter validation
- Documentation standards

## Workflow
1. Scan plugin directory structure
2. Validate against Claude Code schemas
3. Identify and report issues
4. Apply automatic fixes where possible
5. Generate comprehensive report
