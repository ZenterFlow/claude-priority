# Claude Prioritise

## What This Plugin Teaches
This plugin helps developers maintain project context, surface priorities, and manage backlogs persistently across development sessions.

## Skills Included
- **project-status**: Assesses project status, surfaces priorities, and maintains persistent backlog

## Agent
- **claude-prioritise-tutor**: Interactive guide for project status and priority management

## How to Use
1. Type: "What should I work on?" or "Show project status"
2. Review current status and suggested next steps
3. The plugin maintains logs in `persistence/` directory:
   - `log.md`: Continuous activity log
   - `ideas.md`: Backlog of ideas
   - `todo.md`: Current task list
